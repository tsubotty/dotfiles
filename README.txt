This repository is dotfiles of karasuhaneto.  

dotfiles are 
.zshrc
.zsh
.vimrc
.gvimrc 
.gemrc  
.vimperatorrc 
.emacs.d
.gitconfig
etc...

Usage:
Clone from github
git clone https://github.com/karasuhaneto/dotfiles.git

Move into the directory, run "setup.sh", then symblic links from your $HOME to dotfiles are made.
